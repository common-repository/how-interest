== Description ==

A very simple plugin that lets visitors to your site register their interest in your service.  It captures their name, telephone number and email address.

 - Once installed, the plugin creates a widget that you can add to your site.  The widget captures the visitors name, email address and phone number.
 - When a visitors registers their details the site admin is sent an email with the visitors details
 - You can see a list of all visitors who have registered their interest by clicking on the "How-Interest" tab under the "Settings" heading 

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `how-interest.zip` to the `/wp-content/plugins/` directory and unzip it
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag the 'How Interest - Register Interest Widget' to your widget area

== Frequently Asked Questions ==


= Form validation alert boxes do not have any text =

To ensure that validation alerts work,  update the widget options in the Appearence->Widgets section.  It is important that you press Save on the widget options box.
